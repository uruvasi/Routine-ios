# iPhone 実機動作確認 手順

## 事前準備

### 1. iPhone を Mac に接続
- Lightning または USB-C ケーブルで接続
- iPhone に「このコンピュータを信頼しますか？」が表示されたら **「信頼」** をタップ

### 2. Xcode でプロジェクトを開く
- `Routine-ios.xcodeproj` をダブルクリック
- Xcode が起動するまで待つ（初回は数分かかる場合あり）

---

## 実機ビルド手順

### 3. ビルドターゲットを iPhone に切り替える
- Xcode 上部中央のデバイス選択バーをクリック
- 接続した iPhone の名前（例: `uruvasi の iPhone`）を選択

### 4. 署名設定を確認する
- 左サイドバーで `Routine-ios`（青いアイコン）をクリック
- **TARGETS** → `Routine-ios` を選択
- **Signing & Capabilities** タブを開く
- `Automatically manage signing` にチェックが入っていることを確認
- **Team** に自分の Apple ID が表示されていることを確認
  - 表示されていない場合: Xcode メニュー → **Settings** → **Accounts** → `+` で Apple ID を追加

### 5. ビルド・インストール
- ▶︎（Run）ボタンをクリック（または `⌘ + R`）
- 初回は「デベロッパーを信頼」する作業が必要（下記参照）

---

## 初回のみ: iPhone の設定（2ステップ）

### ステップ A — デベロッパモードを有効にする

1. iPhone の **設定** アプリを開く
2. **プライバシーとセキュリティ** をタップ（下の方にあります）
3. 一番下までスクロールして **デベロッパモード** をタップ
4. スイッチを **オン** にする
5. 再起動を求められるので再起動する
6. ロック解除後に「デベロッパモードをオンにしますか？」が出たら **「オンにする」** をタップしてパスコードを入力

> **「デベロッパモード」が見当たらない場合**
> Mac と iPhone をケーブルで繋いだ状態で Xcode を一度起動すると項目が出現します。

### ステップ B — デベロッパ App を信頼する

1. iPhone の **設定** アプリを開く
2. **一般** → **VPNとデバイス管理** をタップ
   - 古い iOS の場合は「プロファイルとデバイス管理」と表示されることがあります
3. **デベロッパ App** の欄にある自分の Apple ID（メールアドレス）をタップ
4. 青字の **「〇〇@xxx.com を信頼」** をタップ
5. ポップアップが出るので **「信頼」** をタップ

> この作業は新しい Apple ID で初めて実機ビルドするときに一度だけ必要です。

### 完了後
Xcode に戻り ▶︎（Run）をクリックするとアプリが起動します。

## 動作確認チェックリスト

### ルーティン作成
- [ ] `+` ボタンで新規ルーティンを作成できる
- [ ] タスクを追加し、時間ピッカーで時間を設定できる
- [ ] 保存後にリストに表示される
- [ ] アプリを終了・再起動してもデータが残っている

### 編集・並び替え・削除
- [ ] ルーティン行タップで編集画面が開く
- [ ] タスクを長押しドラッグで並び替えられる
- [ ] ルーティンを削除できる

### タイマー実行
- [ ] ▶︎ ボタンでタイマー画面が開く
- [ ] 開始ボタン押下でカウントダウンが始まる
- [ ] 開始時にビープ音と TTS（音声読み上げ）が流れる
- [ ] 一時停止 → 再開で時間が正しく引き継がれる
- [ ] ホームボタンでバックグラウンドへ移動 → アプリに戻ると残り時間が補正されている
- [ ] タスク完了時に通知が届く
- [ ] 時間切れで自動的に次のタスクへ進む
- [ ] 全タスク完了後に「完了！」画面が表示される

### 設定
- [ ] 言語切替（日本語 / English）で UI 全体が切り替わる
- [ ] Markdown エクスポートでファイルが出力される
- [ ] エクスポートした `.md` を再インポートでルーティンが追加される
- [ ] リセットで全データが消え、再起動後も空のまま

## トラブルシューティング

| 症状 | 対処 |
|---|---|
| ビルドエラーが出る | Xcode 上部のエラー内容をコピーして共有 |
| iPhone が認識されない | ケーブルを抜き差し → 再度「信頼」をタップ |
| 「No account」エラー | Settings → Accounts で Apple ID を追加 |
| 音が出ない | iPhone の消音スイッチを確認 |
| 通知が来ない | 設定 → 通知 → Routine-ios → 通知を許可 |
